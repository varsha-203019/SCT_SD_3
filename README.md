## Task Title
Sudoku Solver

## Description
This program solves a 9x9 Sudoku puzzle using the backtracking algorithm. Empty cells are represented using 0 and the program fills in the correct values following Sudoku rules.

## Programming Language
C

## How It Works
1. The program scans the grid for empty cells.
2. It tries numbers from 1 to 9.
3. Checks row, column, and 3x3 grid constraints.
4. Uses backtracking to find the correct solution.

## Key Concepts Used
- Backtracking algorithm
- Recursion
- 2D arrays
- Conditional logic

## Output
Displays the solved Sudoku grid.
